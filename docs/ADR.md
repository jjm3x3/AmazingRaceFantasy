## Coding Standards

### Type Script

- Interface in TypeScript should start with an upper case letter, but not with an 'I' unless the word starts w/ 'I'
  - _reason: as per [this SO response](https://stackoverflow.com/a/68504607) it is not typically interfaces in TypeScript do not prefer to use the convention where an 'I' preceded the name, instead it should be clear what the interface is for_
